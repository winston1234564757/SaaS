## 2. Які дані ми збираємо / What Data We Collect

### 2.1. Дані реєстрації / Registration Data

- **Номер телефону** — для авторизації через SMS OTP;
- **Email-адреса** — для технічної ідентифікації облікового запису (може бути згенерована системою);
- **Ім'я та прізвище** — за бажанням, для персоналізації;
- **Фото профілю** — за бажанням.

### 2.2. Дані Майстра / Master Data

- Назва послуг, ціни, робочий графік — для роботи Платформи;
- Telegram Chat ID — для сповіщень (за бажанням);
- Платіжні реквізити (токен картки) — зберігаються виключно у Monobank; BookIT не зберігає повні дані картки.

### 2.3. Дані записів / Booking Data

- Дата, час, послуга, статус запису;
- Нотатки до запису (якщо надані).

### 2.4. Технічні дані / Technical Data

- IP-адреса, тип браузера, дані сесії — для безпеки та аналітики;
- Логи дій — для відлагодження та безпеки.

4.3. Резервні копії зберігаються **7 днів**.

4.4. Передача даних за межі ЄС здійснюється на підставі Стандартних договірних положень ЄС (SCCs).

## 5. Обробка номерів телефонів / Phone Number Processing

5.1. Номери телефонів збираються виключно для автентифікації через SMS OTP.

5.2. SMS-сервіс: повідомлення надсилаються через **TURBOSMS**. Провайдер отримує лише номер телефону і текст повідомлення — без інших персональних даних.

5.3. Спроби підтвердження коду обмежені (10 спроб за 15 хвилин) для захисту від зловживань.

5.4. OTP-коди зберігаються у вигляді хешів і автоматично видаляються після 10 хвилин.

Для реалізації прав надішліть запит на **bookitsupport@gmail.com**. Термін відповіді — **30 днів**.
