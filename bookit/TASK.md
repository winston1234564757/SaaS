Увійди в режим архітектора. Завдання: масштабувати наш секвенсор підказок (useTour та AnchoredTooltip) на АБСОЛЮТНО ВСІ сторінки дашборда майстра. Тур має стати повноцінним "менеджером з успіху клієнтів" (Customer Success), який пояснює цінність кожної фічі.

Вимога до інтеграції:
У кожному з перелічених нижче компонентів:

Виклич хук const { currentStep, nextStep } = useTour('назва_туру', кількість_кроків);.

Обгорни цільові елементи у <div className="relative">.

Рендери <AnchoredTooltip> з абсолютним позиціонуванням, передаючи primaryButtonText="Далі" (або "Зрозуміло" для останнього кроку) та onPrimaryClick={nextStep}.
